💡 Problem Statement

"AI-Powered Public Grievance Classifier"

Citizens often face difficulties in reporting and tracking grievances efficiently. Our solution leverages AI to classify complaints and provide actionable insights to authorities.

⚙️ Features
🤖 AI Chatbot Integration
Classifies user complaints automatically
Provides smart interaction for better issue understanding
📍 Heatmap Visualization
Displays concentration of complaints by area
Helps authorities identify critical zones
🔐 User Authentication System
User Login & Admin Login
OTP-based secure authentication
🛡️ Firewall Database Security
Ensures data protection and secure storage
🌐 Web-Based Platform
Easy-to-use interface for both users and admins
🏗️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: JSON , Python
 AI Integration: Python-based NLP model for grievance classification, LLM-powered chatbot, and real-time text processing
🚀 How It Works
User logs in via OTP authentication
Submits a grievance through chatbot
AI classifies the issue into categories
Complaint is stored securely in the database
Heatmap updates to reflect issue concentration
Admin can monitor and manage complaints
